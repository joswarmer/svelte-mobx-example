## svelte-mobx-example
Reactive MVVM example with [MobX](https://mobx.js.org) & [Svelte](https://svelte.dev) using [svelte-mobx](https://www.npmjs.com/package/svelte-mobx) 

## Get started

```bash
cd svelte-mobx-example
npm install
npm run dev
```

Navigate to [localhost:5000](http://localhost:5000)
